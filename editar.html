Editar rodapé (caixa fixa)

Muda aqui as 4 linhas que aparecem sempre visíveis no fundo do cardápio. A 1ª linha aparece a negrito/dourado, as outras 3 normais. Isto não precisa de código nem do GitHub — só de um token de acesso (explico abaixo como criar).
Ainda não tenho token — como crio um?
  1. Entra no GitHub e vai a Settings → Developer settings → Personal access tokens → Fine-grained tokens.
  2. Clica em Generate new token.
  3. Em "Repository access", escolhe Only select repositories e seleciona cardapio-parque-sao-caetano.
  4. Em "Permissions", abre Repository permissions e muda Contents para Read and write.
  5. Escolhe uma validade (ex: 1 ano) e clica em Generate token.
  6. Copia o token (começa por github_pat_) e cola-o no campo acima. Só tu o vês — fica só no teu aparelho.
Enquanto estiver ativo, o rodapé muda sozinho para o texto abaixo durante o horário indicado, e volta ao texto normal automaticamente fora desse horário — não precisas de fazer nada nessa hora.

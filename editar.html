Editar cabeçalho e rodapé

Muda aqui o texto do rodapé, as fotos que rodam no topo do cardápio e a imagem de publicidade do rodapé. Isto não precisa de código nem do GitHub — só de um token de acesso (explico abaixo como criar).
Esta página foi atualizada entretanto (tens esta aba aberta há algum tempo). Para não corrermos o risco de apagar alterações mais recentes, recarrega a página (F5) antes de continuares a editar ou a guardar.
Ainda não tenho token — como crio um?
  1. Entra no GitHub e vai a Settings → Developer settings → Personal access tokens → Fine-grained tokens.
  2. Clica em Generate new token.
  3. Em "Repository access", escolhe Only select repositories e seleciona cardapio-parque-sao-caetano.
  4. Em "Permissions", abre Repository permissions e muda Contents para Read and write.
  5. Escolhe uma validade (ex: 1 ano) e clica em Generate token.
  6. Copia o token (começa por github_pat_) e cola-o no campo acima. Só tu o vês — fica só no teu aparelho.
Enquanto estiver ativo, o rodapé muda sozinho para o texto abaixo durante o horário indicado, e volta ao texto normal automaticamente fora desse horário — não precisas de fazer nada nessa hora.
Fotos do cabeçalho
Estas fotos aparecem a rodar no topo do cardápio, por trás do nome do restaurante. Define a ordem, quantos segundos cada uma fica visível antes de passar para a seguinte (dissolvem suavemente entre si), e adiciona ou remove fotos.
Publicidade da barra fixa
Podes mostrar uma imagem de fundo (de um anunciante) na barra fixa que fica sempre visível no fundo do ecrã, por baixo do texto de cima — ou deixá-la sem foto, só com o texto sobre fundo liso.

Publicidade no grupo "Vinho da Casa"
Podes mostrar uma foto de publicidade (ex: "Vinho do Mês") no fundo do grupo "Vinho da Casa", dentro da secção de Vinhos & Bebidas. Aparece sempre que o cliente abrir esse grupo, sem horários.

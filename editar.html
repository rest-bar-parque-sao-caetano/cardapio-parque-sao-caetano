Edita os textos e as fotos do cabeçalho e da barra fixa. Guarda com um clique — só precisas do teu token do GitHub.
Ainda não tenho token — como crio?
  1. No GitHub: Settings → Developer settings → Personal access tokens → Fine-grained tokens.
  2. Generate new token.
  3. Em "Repository access": Only select repositories → cardapio-parque-sao-caetano.
  4. Em "Permissions" → Repository permissions → Contents = Read and write.
  5. Generate token, copia (começa por github_pat_) e cola acima.
